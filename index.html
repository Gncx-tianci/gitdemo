Git 常用配置速查表

新手友好 · 即查即用 · 简洁清晰

基础身份配置

  • 全局用户名配置 git config --global user.name "你的用户名"

    设置所有 Git 仓库的提交者用户名，只需配置一次

  • 全局邮箱配置 git config --global user.email "你的邮箱@example.com"

    设置所有 Git 仓库的提交者邮箱，与 Git 平台（如GitHub）邮箱一致更佳

配置查看与修改

  • 查看全局配置 git config --global --list

    列出所有全局配置项，验证用户名/邮箱是否设置成功

  • 单个仓库配置身份 git config user.name "专属用户名"

    去掉 --global，仅为当前仓库设置独立的提交身份（适用于多账号场景）

  • 修改默认编辑器 git config --global core.editor "notepad++"

    将 Git 默认编辑器改为 Notepad++（也可替换为 vscode、vim 等）

实用优化配置

  • 设置彩色输出 git config --global color.ui auto

    让 Git 命令输出带颜色，更易区分不同信息

  • 设置默认分支名 git config --global init.defaultBranch main

    新建仓库时默认分支名设为 main（替代旧的 master）
